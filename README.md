ArcX Studio

ArcX Studio is a powerful Windows application designed for working with Minecraft Legacy Edition archive files.
It provides advanced tools for viewing, extracting, repairing, and rebuilding ARC archives used in console versions of the game.

ArcX Studio focuses on speed, stability, and usability, giving modders and developers a modern interface for managing game assets.

---

✨ Features

📦 Archive Explorer

Browse archive contents with a clean file explorer interface.
Quickly navigate through folders and files stored inside ARC archives.
built-in ".bak" backup file 

🛠 Archive Repair

Automatically detect and repair common archive issues such as:

- broken offsets
- corrupted file entries
- invalid archive structures

🧾 Live Operation Logs

Real-time logging for archive operations including:

- archive building
- archive repairing
- extraction processes

🖼 Texture Viewer

Preview texture files stored inside the archive.

Future versions will include texture replacement and editing tools.

⚙ Settings Manager

Customizable settings stored locally for:

- default extraction paths
- logging preferences
- repair behavior
- performance options

---

📁 Supported Files

ARC Studio is designed to work with files used in Minecraft Legacy Edition archives, including:

- ".arc" archive files
- ".swf" texture containers
- ".fui" texture containers
- ".loc" localization files
- various internal game assets

---

🖥 System Requirements

- Windows 11 Pro
- .NET Framework 4.8
- 64-bit processor recommended

---

🚀 Getting Started

1. Launch ARC Studio
2. Open an ".arc" archive
3. Browse the archive contents
4. Extract, edit, or rebuild files
